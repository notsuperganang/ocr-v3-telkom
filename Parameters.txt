# Notes on Method Outputs

* **`print()` output includes:**

  * `input_path`: `(str)` Input path (image/PDF)
  * `page_index`: `(int \| None)` PDF page index or `None`
  * `model_settings`: `(dict)` Pipeline model settings
  * `use_doc_preprocessor`, `use_seal_recognition`, `use_table_recognition`, `use_formula_recognition`: `(bool)` Enabled sub-pipelines
  * `doc_preprocessor_res`: `(dict)` Results from preprocessor (if enabled)
  * `parsing_res_list`: `(list[dict])` Parsed content (reading order, bboxes, labels, text)
  * `overall_ocr_res`: `(dict)` OCR results with detection polys, scores, recognized texts, etc.
  * `formula_res_list`: `(list[dict])` Formula recognition results
  * `seal_res_list`: `(list[dict])` Seal recognition results
  * `table_res_list`: `(list[dict])` Table recognition results

* **`save_to_json()`**: Saves above content to JSON.

* **`save_to_img()`**: Saves visualizations (layout, OCR, reading order, etc.) as PNG.

* **`save_to_markdown()`**: Saves `.md` files for each page (use directory for PDFs).

* **`concatenate_markdown_pages()`**: Merges multi-page markdown results into one doc.
